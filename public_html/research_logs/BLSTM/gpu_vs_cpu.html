GPU v/s CPU runtime and performance statistics

We can see that the GPU speeds up the performance by about 3 TIMES as compared to the GPU counterpart.

Read the below graphs as follows :
> Error (orange) : The CTC-error at that specific input sample
> Time (green) : Time taken in seconds to forward-backward pass that sample
> Total Time (red) : Total time passed till that point

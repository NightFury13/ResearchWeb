Other Idea :
> Dr. Batra suggessted that as long as we keep using deterministic procedures to pre-process our data, the model can still be gamed. So how about adding random noise to say about 30-40% of the image and then use a denoising-autoencoder on top of it to obtain the rectified image, then apply a deterministic noise-reduction technique and finally pass it to the deep network? This stacked network cannot be gamed using the similar approaches we followed for gaming the previous autoencoder experiments as now the Nash Equilibrium changes and prediction of the output is not possible.

VQA Iterative Algo Adversaries

Creating Fooling Images for the VQA model using iterative fast-sign Algo

The adversarial examples created by fooling the VGG_ILSVRC_19_layers model. The VQA model seems to show some signs of robustness however fails in cases similar to that of fooling images created on CaffeNet.

The below described images all give a fooling class confidence score >= 95% on VGG_ILSVRC_19_layers model.

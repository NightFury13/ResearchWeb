Character-Level Language Model

Trainig a RNN to learn english language model from text-samples.

The RNN is a simple one (non-LSTM nodes). An input text-corpus is fed to the network and the network learns to predict character-by-character, a sentence of english. The hidden layer nodes are update using the tanh non-linearity,

h = np.tanh(np.dot(self.W_hh, self.h) + np.dot(self.W_xh, x))

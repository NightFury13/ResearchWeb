Iterative Algo Adversaries

High fooling class confidence images

The fast-sign algorithm when applied in an iterative fashion yields much better results as compared to the straight forward one-go-with-high-eps value. For any image, a fooling class confidence of atleast 90% (96% on average) can be achieved in about 5 iterations with an eps value as low as 4 (as compared to the 30 or more being used for the single-shot version). The results of this approach can be seen below.

> There are two ways of getting the confidence measures (predictions) of a model on input in caffe.
> Using a net=caffe.Classifier() and then doing a net.predict()
> Using a net=caffe.Net() and then getting values of last layer : net.blobs['prob'].data
> ISSUE : While the second method is fooled as expected from the fooling algo, the first approach seems to be robust! > Link1 Link2 to discussion.

Caffe prediction scheme difference

Different prediction results on caffe.Net() and caffe.Classifier()

There is a difference in the way that the caffe.Net() and caffe.Classifier() prediction schemes work. In caffe.Net() approach, the predictions are obtained by net.blobs['prob'].data.argmax() while in caffe.Classifier() scheme, prediction is done by the wrapper function net.predict([image]) which inturn calls the net.forward_all() method on 10 crops of the input image (oversampling is performed) and the mean prediction is returned.
The oversampling scheme seems to be robust to adversarial perturbations!!


Cat image coverted to Goldfish



> The CaffeNet model predicts this image as class 2 (GoldFish) as expected using the caffe.Net() scheme [No oversampling and mean-crops]
> The CaffeNet model predicts this image as class 287 (Puma) unexpectedly (a class very similar to original base class - 283 Tabby Cat) using the caffe.Classifier() scheme. [Oversampling and Mean-crops]
